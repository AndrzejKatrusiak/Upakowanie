# Dziennik pracy

## 2026-05-18

- Utworzono statyczną aplikację edukacyjną działającą w przeglądarce i nadającą się do publikacji przez GitHub Pages.
- Dodano rzut komórki elementarnej wzdłuż osi `x`, `y`, `z`, operacje symetrii i kopie cząsteczek zgodne z liczbą `Z`.
- Dodano sterowanie: obrót myszą, przesunięcia strzałkami, reset oraz podgląd rozwiązania.
- Dodano workflow publikacji GitHub Pages i lokalny serwer developerski `scripts/dev-server.js`.
- Uzupełniono wymagania po uwagach użytkownika:
  - prowadzony jest ten dziennik pracy,
  - ruch myszy w lewo/prawo odpowiada obrotowi wokół osi pionowej,
  - ruch myszy w górę/dół odpowiada obrotowi wokół osi poziomej,
  - dodano wybór wariantu źródła danych dla związku,
  - dodano widoczne źródła literaturowe i bazodanowe w panelu danych,
  - dodano pionowy suwak skali obrazu.
